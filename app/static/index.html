Mijn Thuislocatie & Radii

Bandplanner berekent de afstandsscore op basis van je thuislocatie. Geef hier de coördinaten van je thuisbasis op.

Zoekstralen (Radius in kilometers)

Zalen buiten deze radius worden uitgesloten voor het betreffende zaaltype.

API Sleutels & Integraties

Configureer hier je API credentials voor externe koppelingen. Als je deze leeg laat, wordt er teruggevallen op het .env bestand.

E-mail Notificaties (SMTP)

Configureer de SMTP server om e-mailnotificaties te ontvangen bij nieuwe concerttips.

Nieuwsbrieven Automatisch Inlezen (IMAP)

Verbind Bandplanner met een specifieke mailbox (bijv. een apart e-mailadres waar je al je poppodium-nieuwsbrieven naar doorstuurt). Alle ongelezen e-mails worden automatisch door Gemini geanalyseerd op concerttips.

iCalendar Kalenderfeed

Kopieer de onderstaande link en voeg deze toe aan je kalenderprogramma (zoals Google Calendar, Apple Agenda of Outlook). Concerten die je markeert als 'Geïnteresseerd' worden automatisch in je agenda gezet!
